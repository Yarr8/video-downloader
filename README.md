# Video Downloader Microservices

Service for downloading videos from different platforms (Reddit only supported for showcase).

## Stack

- Java 21 + Spring Boot
- Python + Flask
- PostgreSQL
- Docker / Docker Compose
- WebClient / REST
- Microservice architecture
